# `switch` to compare an expression with multiple values

```javascript
switch (/* expression */) {
    case /* a value */:
    /* code to run */
    break;

    case /* a value */:
    /* code to run */
    break;

    case /* a value */:
    /* code to run */
    break;

    default:
    /* code to run if there was no match */
}
```
